quasardb
--------

Thank you for using our software!

quasardb is a high-performance, scalable, persistent and peer-to-peer key/value database.

Up to date documentation can be found at this address: https://doc.quasardb.net/

Report bugs to: bug@quasardb.net

INSTALLING THE PYTHON API FROM SOURCE
-------------------------------------

You will need a C++ compiler (g++ or clang++) and the Python development headers.

Then to build and install:

    sudo python setup.py install
